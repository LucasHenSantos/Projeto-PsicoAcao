# PsicoAção: Presence Control System
Uma aplicação web para gerenciar o **check-in e check-out de alunos** durante eventos promovidos pela equipe do PsicoAção, facilitando o controle de participação e presença.

## Funcionalidades Principais
- Registro de entrada (check-in) e saída (check-out) dos alunos.
- Listagem de participantes por evento.
- Controle de frequência e geração de relatórios simples.
- Interface amigável e acessível via web.

## Usabilidade para Coordenadores do Curso de Psicologia

O sistema oferece **acesso exclusivo aos coordenadores do curso de Psicologia**, permitindo realizar as seguintes operações:

- **Cadastrar Eventos:** Criar novos eventos relacionados às atividades do curso.  
- **Cadastrar Professores:** Inserir e gerenciar informações de docentes vinculados aos eventos.  
- **Visualizar Eventos Arquivados:** Consultar eventos anteriores que foram arquivados, mantendo o histórico completo.  
- **Baixar Planilhas:** Exportar dados dos eventos arquivados em planilhas para controle e relatórios.  
- **Gerenciar Datas de Eventos:** Definir datas dos eventos utilizando o calendário integrado ou inserindo manualmente, conforme a necessidade.
